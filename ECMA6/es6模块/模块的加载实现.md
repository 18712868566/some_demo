<h3>1、es6模块用于浏览器中</h3>

标准写法是：

```
<script type="module">
    console.log('module')
</script>
```

按照预期效果是会非堵塞加载，然后再执行。事实上是里面的代码根本不会执行。

又比如说：

```
<script type="module" src="./bar.js"></script>
<script src="./foo.js"></script>
```

你以为会先启用异步加载``bar.js``，然后再加载``foo.js``文件么？

对不起，``bar.js``文件不会被浏览器加载。

原因依然是，不支持。心很累。
